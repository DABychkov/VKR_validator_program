1- Скопируй проект
Скопируй папку проекта с флешки/облака в удобное место на новом компьютере.

2- Создай и активируй виртуальное окружение
Перейди в папку проекта в cmd и выполни:

cmd
python -m venv venv
venv\Scripts\activate.bat


3- Установи зависимости
cmd
pip install -r requirements.txt

4- Запусти приложение
cmd
python app.py

5- Теперь сайт должен работать на новом компьютере по адресу http://127.0.0.1:5000
