1- Скопируйте проект с гитхаб

2- Создайте и активируйте виртуальное окружение
Перейдите в папку проекта в cmd и выполните:

cmd
python -m venv venv
venv\Scripts\activate.bat


3- Установите зависимости
cmd
pip install -r requirements.txt

4- Запустите приложение
cmd
python app.py

5- Теперь сайт должен работать по адресу http://127.0.0.1:5000
